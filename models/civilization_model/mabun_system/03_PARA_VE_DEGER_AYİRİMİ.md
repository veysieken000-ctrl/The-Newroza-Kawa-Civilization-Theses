Ölçünün Ölçülenin Yerine Geçmesi Problemi
GİRİŞ: PARA NEDİR?
Modern ekonomi para üzerine kuruludur.
Ancak çoğu sistem şu hatayı yapar:
➡️ Parayı değer sanır.
Rasterast yaklaşımında para:
amaç değildir
gerçek zenginlik değildir
varlık üretmez
Para yalnızca:
ölçüdür.
Nasıl ki cetvel uzunluk değilse,
para da değer değildir.
I — DEĞERİN ONTOLOJİK TANIMI
Hebûn perspektifinde değer:
varlığı sürdürüyorsa
hayatı destekliyorsa
denge kuruyorsa
değerdir.
Örnek:
gıda → değer
su → değer
emek → değer
bilgi → değer
Para ise:
➡️ bunların temsilidir.
II — ÖLÇÜNÜN BOZULMASI
Sorun şu noktada başlar:
Para:
amaç hâline gelir
birikim nesnesi olur
üretimden kopar.
Bu durumda:
ölçü, ölçülenin yerine geçer.
Sonuç:
para büyür
gerçek değer küçülür.
III — MODERN EKONOMİDE KIRILMA
Modern sistemde:
para paradan doğar
finans üretimden ayrılır
spekülasyon emekten hızlı büyür.
Bu, Mabûn açısından:
➡️ ontolojik tersliktir.
Çünkü:
değer üretmeyen kazanç
varlık dengesini bozar.
IV — PARANIN DOĞAL İŞLEVİ
Mabûn’da para:
✔ değişim kolaylaştırır
✔ değeri ölçer
✔ dolaşımı hızlandırır
ama:
❌ değer üretmez
❌ sınırsız birikemez
❌ hüküm veremez.
Para:
araçtır.
V — DOLAŞIM İLKESİ
Doğada hiçbir şey durmaz:
kan dolaşır
su döner
enerji akar.
Para da:
➡️ dolaşmak zorundadır.
Dolaşım durursa:
ekonomi donar
üretim azalır
toplum katılaşır.
Bu yüzden Mabûn:
Para birikimini değil, dolaşımını esas alır.
VI — PARA BİRİKİMİ NEDEN RİSKLİDİR?
Aşırı birikim:
gücü merkezileştirir
bağımlılık üretir
toplumsal güveni yok eder.
Doğadaki karşılığı:
hücrede tümör
ekosistemde tek tür baskınlığı.
Ekonomik eşitsizlik:
sadece sosyal değil,
➡️ ontolojik dengesizliktir.
VII — FAİZ VE DEĞER SORUNU
Mabûn perspektifinde temel ilke:
Değer üretmeyen artış problemlidir.
Çünkü:
emek olmadan büyüyen kazanç
gerçek üretimden kopar.
Bu:
borç döngüsü
yapay zenginlik
kırılgan sistem üretir.
VIII — PARA VE AHLAK İLİŞKİSİ
Para nötr değildir.
Nasıl kullanıldığı:
ahlaki sonucu belirler.
Ahlaktan kopmuş para:
manipülasyon aracı olur
insanı ölçüye indirger.
Ahlaka bağlı para:
emeği korur
üretimi destekler
dengeyi güçlendirir.
IX — GERÇEK ZENGİNLİK NEDİR?
Mabûn’da zenginlik:
para çokluğu değildir.
Gerçek zenginlik:
üretim kapasitesi
dayanışma
sürdürülebilir kaynak
güven duygusudur.
Para bunların yerine geçtiğinde:
➡️ toplum fakirleşir, sayı büyüse bile.
X — KRİZLERİN ASIL NEDENİ
Finans krizleri çoğu zaman:
para eksikliği değil,
paranın gerçek değerle bağının kopmasıdır.
Bu bağ kopunca:
balonlar oluşur
güven dağılır
sistem çöker.
